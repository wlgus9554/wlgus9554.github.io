## C언어 코드 - 정보처리산업기사 문제

```c
void main (void) {
    int x[] = {100, 200, 300, 400};
    int y;
    y = *(x+2)+50;
    printf("%d", y);
}
```

위의 코드의 출력값을 구하는 문제이다.  
예시로 배열의 주소를 대입해서 구해보자.  
  
int의 크기(byte)는 4byte이다.  

배열 0번 (100)의 주소를 1000이라고 하였을때, 
{ 100(1000), 200(1004), 300(1008), 400(1012) } 가 되고  
1000 + 2 * 4(byte) = 1008 로 계산이 된다.  
따라서 300(1008) + 50 을 계산하면 출력되는 y의 값은 350이 된다.
