## 버블 정렬
**예시**
[ 5, 9, 4, 1, 3]<br>

**1회전** <br>
5, 4, 9, 1, 3<br>
5, 4, 1, 9, 3<br>
5, 4, 1, 3, 9<br>
- 가장 큰 수(9)가 맨뒤로 갈때까지 비교 -> 교환한다<br>
**2회전** <br>
4, 5, 1, 3, 9<br>
4, 1, 5, 3, 9<br>
4, 1, 3, 5, 9<br>
- 2번째 큰 수(5)가 뒤에서 2번째로 갈때까지 비교 -> 교환한다<br>


## 선택 정렬
**예시**
[ 5, 9, 4, 1, 3]<br>

**1회전** <br>
1, 9, 4, 5, 3<br>
- 가장 낮은 숫자를 맨 앞자리랑 교환한다.<br>
**2회전** <br>
1, 3, 4, 5, 9<br>
- 그 다음 낮은 숫자랑 2번째 자리랑 교환한다.<br>

## 삽입 정렬
**예시**
[ 5, 4, 1, 9, 3]<br>

**1회전** <br>
4, 5, 1, 9, 3<br>
- 맨앞 5와 4를 비교해서 낮은 숫자를 앞으로 불러온다.<br>
**2회전** <br>
4, 1, 5, 9, 3<br>
1, 4, 5, 9, 3<br>
- 5와 1을 비교해서 낮은 숫자를 앞으로 불러온다.<br>
- 맨앞에 4와 1을 비교해서 낮은 숫자를 앞으로 불러온다.<br>
