## 연산자 (OP-Code)의 기능
- 함수 연산 기능
- 전달 기능
- 제어 기능
- 입출력 기능

## 마이크로프로세서(Microprocessor)
- CPU(중앙 처리 장치)의 기능을 수행하기 위하여 만든 고밀도 집적 회로(LSI)로 **연산 장치, 제어 장치, 레지스터**로 구성

## DOS 내외부 명령어
- 내부 : DIR, PATH, DEL, VER, COPY, DATE, CLS, MD, CD, RD 등
- 외부 : ATTRIB, FROMPT, CHKDSK, FDISK, LABEL, SCANDISK, DISKCOPY, SORT, SYS, UNDELETE, XCOPY

## 정보 통신 시스템의 4대 구성 요소
- 단말 장치
- 전송 회선
- 통신 제어 장치
- CPU

## RS-232C 커넥터 주요 핀의 기능

| 핀번호  | 핀 이름 | 기능 |
| 2 | TXD | 송신 |
| 3 | DCE | 수신 |
| 4 | RTS | 송신 요청 |
| 5 | CTS | 신호 |
| 6 | DSR | DEC의 동작 상태 |
| 7 | SG | 신호 접지 |

## 컴퓨터에 하드 디스크를 새로 장착하고 부팅 가능한 하드 디스크로 만드는 도스 명령어
- FORMAT C: /S => 시스템 파일을 복사하여 부팅 가능한 디스크로 만듦

## 모뎀(변복조기 : MODEM) : 변조(Modulation)와 복조
- (DEModulation)의 합성어로 변조는 디지털 신호를 아날로그 신호로 변환하는 과정, 복조는 아날로그 신호를 디지털 신호로 변환하는 과정을 의미함

## 위성 통신의 다원 접속 방법
- 주파수 분할 다원 접속
- 시분할 다원 접속
- 코드 분할 다원 접속
- 공간 분할 다원 접속

## 패리티 검사
- 패리티 검사에서 전송 중 짝수의 에러 비트가 발생할 경우 에러 검출이 가능하지 않음
- 패리티 검사는 주로 저속 비동기 방식에 이용됨
- 패리티 비트는 짝수(Even) 혹은 홀수(Odd) 패리티로 사용됨
- 패리티 검사를 통하여 전송 신뢰를 높일 수 있음

## 전자파(Electron wave)
- 전자기파(Electromagnetic wave)로, 세기의 변화가 주기적으로 일어나는 전자기장이 공간을 통해 전파해 나가는 것으로 전파(**라디오파**)라고도 함

## Cache Memory(캐시 메모리)
- CPU와 주기억 장치 사이에 있는 고속의 버퍼 메모리로 자주 참조되는 데이터나 프로그램을 메모리에 저장, 메모리 접근 시간을 감소시키기 위한 목적으로 사용됨

## LAN의 특징
- 광대역 전송 매체의 사용으로 고속 통신이 가능하며 에러율이 낮음
- 데이터, 음성, 화상 등의 종합적인 정보 전송이 가능
- 정보 기기의 재배치 및 확장성이 우수
- 자원의 공유와 데이터의 일관성
